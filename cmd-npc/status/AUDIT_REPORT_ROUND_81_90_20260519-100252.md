## Detailed evidence per round

### Round 81 — pet_master pettable coherence

**Bug:** Probe shows 0/389 generated `pet_master` NPCs have `pettable=True`. Brief defines pet_master as the NPC type responsible for pet bonding — semantically must have pettable=True. Source: `gen_one_npc()` set `pettable = npc_type == "monster" and tier <= 5 and rng.random() < 0.20` (only monsters), missing-types branch hardcoded `"pettable": False`.

**Fix:**
- `gen_one_npc()`: `if npc_type == "pet_master": pettable = True; else: pettable = npc_type == "monster" and tier <= 5 and rng.random() < 0.20`
- `extend_to_target()` missing-types branch: `"pettable": t == "pet_master"` and add pet fields after dict when pettable.

**Post-fix:** 389/389 pet_master have `pettable=True` + pet_base_hp/atk/loyalty_init/evolution_path set per brief line 826-838.

### Round 82 — transaction_log cap 3

**Bug:** `cmd-npc/transaction_log/` accumulated 6 build files (002705, 011348, 012526, 014010, 014139, 015224). No sweep policy — disk growth across builds.

**Fix:** Append to `sweep_old_npc_artifacts()`:

```python
tx_dir = REPO_DIR / "cmd-npc" / "transaction_log"
all_tx = sorted(tx_dir.glob("*-template_ship-*.json"))
if len(all_tx) > 3:
    tx_cold = REPO_DIR / "cmd-npc" / "transaction_log-archived"

```

**Post-fix:** Active tx_log = 3 most recent + cold archive folder populated.

### Round 85 — Gen element stat skew alert

**Bug:** Probe shows gen NPC `tâm` element has avg atk 132.8 vs other 5 elements avg 116.5 — a +14% skew. Code review: no element-based boost in `compute_stats()` or `gen_one_npc()`. Skew is statistical artifact from seed=42 + tier/type correlation with element pick (Mersenne Twister state).

R79 says ngũ hành design is element-neutral; the variance is not by design but not actively balanced either.

**Fix:** Add `alert_gen_element_stat_skew()` LOW severity. Computes per-element avg atk and overall; reports any element deviating >10% with per-element table.

**Post-fix:** Alert file `LOW-20260519-100252-npc_gen_element_stat_skew.json` ships with per-element stats + tâm flagged. Remediation: designer may add element-balanced tier-weighted assignment or accept variance ≤15%.
