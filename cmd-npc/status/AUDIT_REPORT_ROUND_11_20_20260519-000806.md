### Round 11 — Protagonist Trần Long NOT unique (R83)

**Bug:** Em chỉ check `"Trần Long" in name` substring. Generated idx=9391 random sinh ra name = "Trần Long" exactly → collision với protagonist idx=1 "Trần Long (player default)". R83 lock: protagonist là duy nhất.

**Evidence:**
```python
sum(1 for n in npcs if n['name'] == 'Trần Long')  # = 1 pre-fix
# idx=9391 name='Trần Long' era=le is_player=None  # phantom protagonist
```

**Fix:** 
1. `PROTECTED_NAMES = {"Trần Long", "Trần Long (player default)", "Sư Vạn Hạnh"}`
2. `vi_name()` retry 8 lần nếu name in PROTECTED_NAMES, fallback append numeric

**Post-fix:** 0 NPC exactly named "Trần Long" trong toàn registry; chỉ 1 entry "Trần Long (player default)" (idx=1 protagonist) ✓
