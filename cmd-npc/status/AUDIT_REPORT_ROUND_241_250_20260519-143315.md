## 🎯 BUG REAL phát hiện và fix (R241)

**Phương án 10 I4 invariant**: Nguyễn era 0 historical figures (other eras: ly=7, tran=7, le=6, tay_son=6, nguyen=**0**).

**Root cause:** `extend_to_target()` inject condition was `if era_counts[era_key] < 5` — chỉ check population count, không check historical-figure count. Existing có 10 Nguyễn NPCs (kiểu "Vua Long Triều Nguyễn (lấy cảm hứng)") — không flag historical → era_counts[nguyen]=10 ≥5 → key_figures inject SKIPPED → 0 historical figures Nguyễn.

**Fix:** Expand condition to include historical-figure-count check:
```python
hist_counts_existing = Counter(
    e.get("era") for e in existing_normalized if e.get("is_historical_figure")
)
needs_population = era_counts[era_key] < 5
needs_historical = hist_counts_existing.get(era_key, 0) < 5
if needs_population or needs_historical:
    # inject key_figures
```

**Post-fix:** Nguyễn era has 6 historical figures (Gia Long, Minh Mạng, Tự Đức, Phan Bội Châu, Phan Châu Trinh, Trương Công Định).

**Bonus fix:** R13 regex forbidding non-diacritic names had "Gia Long" listed, but "Gia Long" IS the canonical Vietnamese reign name with no diacritics needed (unlike "Lê Lợi" / "Nguyễn Huệ" where diacritics matter). Removed "Gia Long" from forbidden pattern.
